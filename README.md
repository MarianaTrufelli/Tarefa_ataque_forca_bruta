# Desafio: Ataque de Força Bruta SMB com Medusa (`Kali Linux`)

Este repositório traz a documentação passo a passo do exercício prático de cibersegurança, utilizando técnicas de força bruta com Medusa e validação via SMB em ambiente controlado.

## Conteúdo

- Relatório detalhado (`relatorio.md`)
- Wordlists exemplares de usuários e senhas (`/wordlists`)
- Comandos e explicações para reprodutibilidade

### Resumo

> Detalhes completos com comandos, objetivos e respostas no arquivo `relatorio.md`.
>
> **Resumo dos principais passos**:
>
> 1. **Preparar ambiente:**
>    - Kali Linux: atacante
>    - Metasploitable2: alvo vulnerável
> 2. **Criar wordlists:**
>    - `usuarios.txt` e `senhas.txt`
> 3. **Executar ataque com Medusa:**
>    - Força bruta SMB
> 4. **Validar acesso via smbclient**
> 5. **Análise crítica de resultados e melhores práticas**
